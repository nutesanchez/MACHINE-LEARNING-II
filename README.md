# Machine Learning II

Magíster en Data Science  
Universidad de Las Américas (UDLA)

## Descripción del repositorio

Este repositorio contiene el desarrollo de las actividades y el proyecto final de la asignatura
**Machine Learning II**, correspondiente al programa del Magíster en Data Science de la
Universidad de Las Américas (UDLA).

La asignatura se centra en el estudio y aplicación de **modelos de clasificación supervisada**,
abordando enfoques lineales y no lineales aplicados a problemas reales, con especial énfasis en la
comparación de modelos bajo un mismo esquema de datos y preprocesamiento.

Los contenidos desarrollados incluyen:
- Preprocesamiento adecuado de datos
- Selección y ajuste de modelos
- Validación cruzada
- Evaluación rigurosa mediante métricas apropiadas para clases desbalanceadas
- Comparación crítica entre distintos algoritmos
- Interpretabilidad y uso responsable de modelos

Las actividades están estructuradas de manera progresiva, utilizando en varios casos un
**proyecto base común**, lo que permite analizar el desempeño, los supuestos y la aplicabilidad de
distintos modelos de aprendizaje automático en contextos reales, particularmente en el ámbito de
la salud.
